# Receipt Tracker

OCR-powered expense tracking application built with Python, Streamlit, EasyOCR, and Pandas

## Live Demo
🔗 https://receipt-tracker-xfjgadgbqzsxjc4jnqbwbe.streamlit.app

## Features
- Upload receipt images
- OCR text extraction
- Automatic expense categorization
- Budget monitoring
- Expense history tracking

## Tech Stack
- Python
- Streamlit
- EasyOCR
- Pandas
- NumPy
